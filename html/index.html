Što je Pazar?

Pazar je besplatna web stranica za objavljivanje oglasa domaćih proizvođača poljoprivrednih proizvoda.

Zašto Pazar?

Pazar cilja informirati građane o kvalitetnim domaćim proizvodima lokalnih poljoprivrednika. Korištenje Pazara poptuno je besplatno, potrebno je samo stvoriti korisnički račun. Moguće je objaviti neograničen broj oglasa. Pazar djeluje samo kao platforma za oglašavanje - svu kupovinu proizvoda potrebno je dogovoriti izravno s prodavačem.

Više o Pazaru

Pazar je napravljen kao završni projekt za naprednu razinu tečaja programiranja Edit CodeSchool 2022. Napravio ga je Jakov Jozić samostalno uz pomoć mentora Edita. Potpuni kod stranice nalazi se na ovom linku.
